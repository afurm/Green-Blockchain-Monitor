## Overview

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up environment variables:
   ```
   OPENAI_API_KEY=your_api_key_here
   ETHERSCAN_API_KEY=your_etherscan_api_key_here
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```
